Zoltar Monorepo

This monorepo contains three components. Project documentation is available at augurproject.github.io/zoltar .

  • Zoltar Contracts — Zoltar whitepaper
  • Statoblast contracts — Statoblast whitepaper
  • Zoltar and Statoblast user interface

Zoltar maintains the shared registries for questions, REP tokens, and universe forks. It is neither a prediction market nor an outcome-reporting oracle. A higher-level protocol can initiate a paid Zoltar fork after an eligible question ends; that fork changes the universe used by every application that chooses to continue from the forked universe.

Statoblast is a binary prediction-market protocol built on Ethereum and Zoltar. It creates security pools, issues transferable outcome shares, coordinates local outcome reporting, and resolves or migrates markets. Secondary-market trading is outside the current protocol.
